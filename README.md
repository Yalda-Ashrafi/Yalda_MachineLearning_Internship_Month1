## 📊 Task Details

### Task 1: Email Spam Classification
- **Dataset:** Spam Dataset (Kaggle)  https://www.kaggle.com/datasets/tmehul/spamcsv
- **Goal:** Classify SMS messages as spam or not spam using machine learning models.  
- **Approach:** Preprocessing text data, feature extraction, and training classifiers.  

### Task 2: Handwritten Digit Recognition (MNIST)
- **Dataset:** MNIST in CSV (Kaggle)  https://www.kaggle.com/datasets/shivambaldha/mnist-dataset
- **Goal:** Recognize handwritten digits (0–9) from pixel values.  
- **Approach:** Logistic Regression, Random Forest, and Linear SVM models with accuracy comparison.  

---

## 📈 Results
- **Logistic Regression:** Accuracy = 0.8875  
- **Random Forest:** Accuracy = 0.92  
- **Linear SVM:** Accuracy ≈ 0.79
